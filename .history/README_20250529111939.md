
# Bank Sim (150 %)

Симуляционная модель банковской системы — курсовая работа.

## Установка

```bash
python -m venv venv
.\venv/bin/activate      # Windows: venv\Scripts\activate
pip install -e .
bank initdb
```

## Использование

```bash
bank register "Иван Иванов"
bank open 1
bank deposit 1 1000
bank history 1
```

## Тесты

```bash
pytest
```
