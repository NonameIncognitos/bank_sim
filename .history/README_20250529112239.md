
# Bank Sim 

Симуляционная модель банковской системы — курсовая работа.

## Установка

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -e .
bank initdb
```

## Использование

```bash
bank register "Иван Иванов"
bank open 1
bank deposit 1 1000
bank history 1
```

**ИЛИ**
введите
```bash
bank shell
```
это откроет интерактивный терминал
Не требует указания bank
